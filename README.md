# SafeStay Student

Student accommodation platform with verified listings and scam reporting.

## Features
- Student, Landlord, and Admin login
- Search and filter accommodation
- Verified badge system
- Review and rating system
- Report scam feature
- Booking management

## Technologies
- HTML, CSS, JavaScript
- Node.js with Express
- SQLite database
